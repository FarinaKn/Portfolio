Web Design

Childhood Escape

2024

A responsive web design project developed as part of my studies. The assignment was to create a website inspired by a favourite place from childhood, which led me to develop a playful, childlike visual style. All illustrations and visual elements were designed by me from scratch in Adobe Illustrator. Built with HTML, CSS and JavaScript, the project focuses on responsive design and a smooth user experience.

Concept & Strategy

The brief was to design a website inspired by a personal childhood memory. The beach became the canvas — a space tied to summer holidays, salt air and the freedom of being young. Illustration was chosen as the primary visual language to capture that handcrafted, nostalgic quality that photography alone couldn't achieve.


Every element — from the ice cream cones to the sandcastles — was drawn from scratch in Adobe Illustrator, then woven into a fully responsive layout built with HTML, CSS and JavaScript. The result is a site that feels alive, warm and genuinely playful without relying on stock assets or templates.
