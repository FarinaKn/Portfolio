Web Design

Childhood Escape

2024

A responsive web design project developed as part of my studies. The assignment was to create a website inspired by a favourite place from childhood, which led me to develop a playful, childlike visual style. All illustrations and visual elements were designed by me from scratch in Adobe Illustrator. Built with HTML, CSS and JavaScript, the project focuses on responsive design and a smooth user experience.
